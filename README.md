Selamat datang di **Aplikasi Singular Value Decomposition dalam Fitur Noise Cancelling dan Mode Transparansi pada Earphone Modern** oleh Kaindra. Proyek ini bertujuan untuk mengimplementasikan metode **Singular Value Decomposition (SVD)** dalam memodelkan fitur noise cancelling dan mode transparansi menggunakan Python.

## ğŸ“– Deskripsi

Fitur **noise cancelling** dan **mode transparansi** pada earphone modern memerlukan pemrosesan suara yang presisi untuk memberikan pengalaman mendengarkan yang lebih personal. Dengan menggunakan metode **Singular Value Decomposition (SVD)**, proyek ini memisahkan sinyal utama dari noise sehingga:

- Noise cancelling dapat mengurangi kebisingan lingkungan.
- Mode transparansi dapat meningkatkan suara sekitar tanpa mengurangi kualitas audio utama.

Proyek ini dirancang menggunakan **Python** dengan modul-modul seperti `NumPy` dan `SciPy` untuk komputasi matriks yang efisien.

---

## ğŸ’» Fitur

1. **Noise Cancelling**:
   - Mengurangi noise dari file audio input.
   - Meningkatkan kualitas suara utama.
2. **Mode Transparansi**:
   - Menggabungkan suara lingkungan dengan audio utama.
   - Simulasi mendekati fitur earphone modern, namun menggunakan pre-recorded audio.
